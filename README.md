# nodeRap
一个简单的前端接口系统。

## 技术概述
前端采用vue+webpack实现，后端使用node+express+mongoDB实现。

## 安装使用

npm install -D

开发环境启动：

npm run dev

由于目前前后端统一启动，但只做了前端文件的动态编译，如果改了后台文件，请重新启动脚本
